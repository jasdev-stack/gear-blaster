Gear Blaster
Battle waves of enemies in this fast-paced arcade shooter!

Play the game
Download the ZIP file, extract it, then open:
build/index.html

Features
- Fast-paced action gameplay
- Multiple enemy types
- Endless Mode

Controls
- Left Arrow/A : Left Movement
- Right Arrow/D : Right Movement
- Mouse Left-click/Space : Shoot energy blaster
- R : Recharge/Reload
